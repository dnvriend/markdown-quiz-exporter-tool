Display help information for markdown-quiz-exporter CLI commands.

## Usage

```bash
# Show general help
markdown-quiz-exporter --help

# Show command-specific help
markdown-quiz-exporter COMMAND --help

# Show version
markdown-quiz-exporter --version
```

## Arguments

- `COMMAND` (optional): Specific command to get help for
- `--help` / `-h`: Show help information
- `--version` / `-v`: Show version information

## Examples

```bash
# General help
markdown-quiz-exporter --help

# Command help
markdown-quiz-exporter search --help

# Version information
markdown-quiz-exporter --version
```

## Output

Displays usage information, available commands, and options.
